
hybkit API
==================================

    The hybkit API provides a Python3 module implementation for interaction with hyb format
    data as python objects. 

    ============================= =====================================================
    :module:`hybkit`              Main classes and functionality of hybkit
    :module:`~hybkit.settings`    Constants and settings information for hybkit classes
    :module:`~hybkit.type_finder` Class for flexible identification of segment type 
                                  from reference identifiers
    :module:`~hybkit.analysis`    Classes for analysis of hyb records
    :module:`~hybkit.plot`        Functions for plotting hyb records
    :module:`~hybkit.util`        Support methods for executable scripts
    ============================= =====================================================
